# Командлеты для сертификатов и проверки подлинности в Lync Server 2013

 

_**Дата изменения раздела:** 2016-12-08_

Командлеты сертификатов и проверки подлинности охватывают широкий диапазон задач, включая управление сертификатами сервера и клиента, управление ПИН-кодами (персональными идентификационными номерами) пользователей и управление доменами SIP и учетными записями Kerberos, используемыми в службах IIS.
