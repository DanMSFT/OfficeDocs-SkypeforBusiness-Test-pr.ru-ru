# Командлеты подключения ТСОП

 

_**Дата изменения раздела:** 2016-12-08_

Корпоративная голосовая связь поддерживает исходящие и входящие звонки через ТСОП без ущерба для качества обслуживания (QoS). Для настройки параметров звонков этого типа можно использовать командлеты Командная консоль Lync Server.
