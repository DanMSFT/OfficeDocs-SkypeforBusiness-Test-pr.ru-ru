# Командлеты федерации и внешнего доступа в Lync Server 2013

 

_**Дата изменения раздела:** 2016-12-08_

Федерация и внешний доступ предоставляют две важных возможности: федерация позволяет пользователям взаимодействовать с людьми за пределами вашей организации, а внешний доступ позволяет пользователям подключаться к Microsoft Lync Server 2013 извне внутренней сети. Командлеты системы Lync Server 2013 для управления федерацией и внешним доступом позволяют вам определить, с кем могут (и не могут) взаимодействовать ваши пользователи и могут ли они подключаться к Lync Server без входа во внутреннюю сеть.

## Командлеты федерации и внешнего доступа

Большинство связанных с федерацией и внешним доступом задач управления можно выполнить из панели управления Lync Server. Эти же задачи можно выполнить с помощью командлетов из командной консоли Командная консоль Lync Server или из сценария.
