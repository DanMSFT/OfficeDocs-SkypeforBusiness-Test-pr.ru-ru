# Командлеты архивирования и мониторинга

 

_**Дата изменения раздела:** 2016-12-08_

Командлеты архивации и мониторинга позволяют администраторам управлять архивацией сеансов обмена мгновенными сообщениями и конференц-связи, регистрировать сведения о вызовах и отслеживать Microsoft Lync Server 2013 с использованием компонента качества взаимодействия (QoE).
