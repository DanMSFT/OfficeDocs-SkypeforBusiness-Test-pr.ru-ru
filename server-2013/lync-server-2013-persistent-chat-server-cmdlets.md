# Командлеты сервера сохраняемого сеанса беседы

 

_**Дата изменения раздела:** 2012-06-27_

Командлеты сохраняемого чата позволяют управлять и настраивать службу сохраняемого часта Microsoft Lync Server 2013 (которая ранее называлась службой группового чата). Сохраняемый чат позволяет пользователям участвовать в интерактивных сеансах обмена мгновенными сообщениями. Хотя эти сеансы могут проходить в реальном времени, содержимое каждого сеанса сохраняется, то есть беседы могут возобновиться любым пользователем в любое время.
