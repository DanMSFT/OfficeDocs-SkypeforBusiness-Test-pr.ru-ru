# Командлеты голосовых приложений

 

_**Дата изменения раздела:** 2016-12-08_

Голосовые приложения выполняются в службе приложений на сервере переднего плана. Эти приложения помогают вам управлять входящими и исходящими телефонными звонками, позволяя вам парковать звонки, работать с неназначенными номерами, координировать работу с единой системой обмена сообщениями Exchange и запускать приложение "Группа ответа".
